1.CLONE A REPOSITORY 

--->git clone https://some link.com 

2.BRANCHING & MERGING 

i)BRANCHING:
	
	WE CAN HAVE MANY BRANCHES TO MASTER .

TO CREATE A BRANCH 

--->git branch Branch1 

TO SEE ALL BRANCHES (ONE WITH * IS OUR CURRENT BRANCH)

--->git branch

SAVE CHANGES by add , AND COMMIT .

CHANGES MADE IN BRANCHES ARE NOT DIRECTLY INTO MASTER 

NOW TO GO MASTER BRANCH  (i.e CHANGE BRANCH BY)

--->git checkout master

MERGE BRANCH1 WITH MASTER BRANCH

--->git merge Branch1
